The Amazon Sales Dashboard is an interactive Business Intelligence solution built using Power BI to analyze Amazon product sales, customer behavior, pricing trends, discounts, and category performance. The dashboard transforms raw sales data into actionable insights using KPIs, dynamic visualizations, slicers, and DAX measures.

This project helps business users monitor overall sales performance, identify high-performing product categories, analyze discount effectiveness, and understand customer purchasing behavior across different cities, age groups, and price ranges.
